# beneath_the_files

You've got a new client! Seems like there's a pesky virus hiding in this computer.  

To get rid of the virus, you must click the files on the desktop and type a short password to check them.  
Click on the virus as it comes to take the files as you investigate.  
Learn the truth about the owner, beneath the files... What darkness lies beneath the piles of unmanaged files and folders?  

**Warnings!**
- Small amounts of blood  
- Horror imagery and descriptions  

<br />
A linear Japanese-inspired horror game with pixel visuals, created during Hack Club's Campfire.  
All assets are original unless otherwise stated (refer to credits.txt).
